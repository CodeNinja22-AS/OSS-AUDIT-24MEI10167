## 📁 Repository Contents

This repository contains the following shell scripts:

- `script1.sh` — System Identity Report  
- `script2.sh` — FOSS Package Inspector  
- `script3.sh` — Disk and Permission Auditor  
- `script4.sh` — Log File Analyzer  
- `script5.sh` — Open Source Manifesto Generator  

---

## ⚙️ Script Descriptions

### Script 1 — System Identity Report
Displays system-related information such as OS, kernel version, current user, uptime, and date/time.

### Script 2 — FOSS Package Inspector
Checks if a given package is installed and displays details using system package managers.

### Script 3 — Disk and Permission Auditor
Analyzes important system directories and shows permissions, ownership, and size.

### Script 4 — Log File Analyzer
Reads a log file (`HealthApp.log`) and counts occurrences of a keyword (default: error).

### Script 5 — Open Source Manifesto Generator
Generates a personalized open-source philosophy statement based on user input.
